We are seeking a Network Security Engineer with strong expertise in load balancers, next-generation firewalls, and enterprise routing and switching. The selected professional will be responsible for implementing change activities, improving infrastructure performance, and supporting perimeter security operations.



Key Responsibilities

Execute changes on F5 Networks load balancers (e.g., configuration updates, new profiles, routing rules).
Implement changes on CheckPoint Next Generation Firewalls (e.g., creation of new security policies).
Manage and configure enterprise network devices: Cisco 7600, Cisco Catalyst 3750, Dell switches, Juniper MX80.
Create and manage VLANs, trunk configurations, routing adjustments, and L2/L3 network components.
Ensure the reliability and performance of network and perimeter security infrastructure.
Collaborate closely with technical teams to support solution evolution and advanced troubleshooting.
Follow internal change management processes and ensure up-to-date technical documentation.
Provide availability for on-call service to support operational continuity.


Mandatory skills

F5 Networks – Load Balancers
CheckPoint Next Generation Firewall
Cisco Routing & Switching Operations


Skills and Qualifications:

Hands-on experience managing enterprise network and security infrastructures.
Strong understanding of networking protocols: TCP/IP, VLANs, routing, switching.
Ability to operate effectively across multi-vendor environments (Cisco, Dell, Juniper).
Experience with change execution in high-availability environments.
Strong analytical and troubleshooting capabilities.
Fluency in English and Italian.
